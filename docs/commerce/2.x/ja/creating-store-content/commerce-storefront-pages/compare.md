# 比較

[Compare] ページは、購入者が比較のために2つ以上の商品を並べて表示できるページです。 [Compare]ページは、購入者が[Catalog]ページで2つ以上の商品を選択した場合にのみ読み込まれる *[Product Comparison Table]* ウィジェットを使用します。
